## 📝 관리 및 수정 방법

### 1. 수정하기
VS Code에서 `frontend/public/leekwon.html` 파일을 열어 내용을 수정합니다.
(React가 아닌 **표준 HTML** 파일이므로, `class` 대신 `class` 속성을 사용하며, Tailwind CSS는 CDN으로 로드됩니다.)

### 2. 반영하기 (GitHub Push)
수정한 내용은 GitHub에 푸시해야 위 공유 링크에 반영됩니다. 터미널에서 아래 명령어를 실행하세요.

```bash
git add frontend/public/leekwon.html
git commit -m "Update profile page content"
git push
```

> **참고:** 푸시 후 변경 사항이 링크에 반영되기까지 약 1~5분 정도 소요될 수 있습니다.

### 3. 정식 배포 시 접근
프로젝트가 Vercel 등에 정식 배포되면, 아래와 같은 도메인 경로로도 접근 가능합니다.
- `https://[도메인주소]/leekwon.html`
